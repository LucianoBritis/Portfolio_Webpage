Front-End Developer

If you are a very first learner about front end development the foremost question of your mind will be how to become a front end developer. Before getting the answer my suggestion is to keep eyes on this article. If you fix yourself to see you as a front end developer within this year. The skills you must master are listed in the below.

  1. HTML / HTML5: Semantic web
  2. CSS / CSS3: LESS, SASS, Media Queries
  3. JavaScript: jQuery, AngularJS, Knockout, etc.
  4. Compatibility quirks across browsers
  5. Angular.js

Full Stack Developer

The idea of a “full-stack developer” isn’t about being fluent in every possible technology there is, because specialization exists for a reason. It’s more about having an understanding in each of the areas below, to communicate intelligently between team members and to be a good asset if the situation needs it.

  1. HTML5, CSS.
  2. Ios.
  3. Android.
  4. Mongodb.
  5. Scala.
  6. NodeJS.
